Yêu cầu:
Tài khoản Github
Trang web tĩnh
Cách sử dụng:
Tạo repository mới trên Github:
Truy cập trang web Github
Nhấp vào nút "New"
Chọn "Repository"
Nhập tên repository
Chọn "Public"
Nhấp vào nút "Create repository"
Cấu hình Github Pages:

Truy cập repository bạn vừa tạo
Nhấp vào tab "Settings"
Chọn "Pages"
Chọn "Source"
Chọn "Branch"
Chọn "master"
Nhấp vào nút "Save"
Deploy trang web:

Push code của trang web của bạn lên branch "master" của repository
Github Pages sẽ tự động deploy trang web của bạn

Lưu ý:

Github Pages chỉ hỗ trợ deploy trang web tĩnh.
Trang web của bạn phải được lưu trữ trong repository Github.
Bạn cần push code của trang web của bạn lên branch "master" của repository.
Hướng dẫn chi tiết:

1. Tạo repository mới trên Github:

Truy cập trang web Github
Nhấp vào nút "New"
Chọn "Repository"
Nhập tên repository (ví dụ: my-website)
Chọn "Public" (để trang web của bạn có thể truy cập được bởi mọi người)
Nhấp vào nút "Create repository"

2. Cấu hình Github Pages:

Truy cập repository bạn vừa tạo (ví dụ: https://github.com/your-username/my-website)
Nhấp vào tab "Settings"
Chọn "Pages"
Chọn "Source"
Chọn "Branch"
Chọn "master" (để deploy trang web từ branch "master")
Nhấp vào nút "Save"

3. Deploy trang web:

Push code của trang web của bạn lên branch "master" của repository
Github Pages sẽ tự động deploy trang web của bạn
Ví dụ:

Bạn có một trang web tĩnh được lưu trữ trong thư mục /my-website trên máy tính của bạn.
Bạn muốn deploy trang web này lên Github Pages.
Thực hiện:

Mở terminal và chuyển đến thư mục /my-website.
Gõ lệnh sau để push code của trang web lên branch "master" của repository:
git add .
git commit -m "Deploy website"
git push origin master
Github Pages sẽ tự động deploy trang web của bạn.
Sau khi deploy thành công, bạn có thể truy cập trang web của bạn tại URL sau:
https://your-username.github.io/my-website
